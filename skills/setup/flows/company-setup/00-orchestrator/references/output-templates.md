# Templates de Output — Context Files

> Cada bloco do Setup gera um arquivo de contexto. Use os templates abaixo como estrutura exata ao salvar cada arquivo. Ao final, os 3 arquivos automáticos (PLATFORM_SPECS, HOOK_LIBRARY, WORKFLOW_STATE) também seguem templates aqui definidos.


Dados padrão por plataforma (dimensões canônicas):
- **Instagram:** 1080×1350 (portrait) | 1080×1080 (square) | 1080×1920 (stories/reels)
- **LinkedIn:** 1080×1080 (quadrado) | 1200×628 (landscape) | 1080×1350 (portrait)
- **TikTok:** 1080×1920 (vertical)
- **YouTube:** 1280×720 (thumbnail) | 2560×1440 (banner)
- **X/Twitter:** 1200×675 (landscape) | 1080×1080 (square)
- **Blog:** sem dimensão padrão — priorizar legibilidade

Para tokens visuais da marca (cores, fontes, gradients): consultar `~/.claude/skills/setup-visual-identity/references/token-extraction.md`.
