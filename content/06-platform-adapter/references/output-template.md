## Notas de uso

- **Mínimo viável:** OUTPUT_ADAPTER.md mínimo precisa ter: Contexto da Sessão + pelo menos 1 plataforma completa com conteúdo + horário sugerido.
- **Plataformas não adaptadas:** se o usuário pediu adaptação para 2 plataformas, incluir apenas as 2 seções relevantes. Não gerar seções vazias.
- **Downstream compatibility:** a Skill 6 (Editorial Calendar) lê o OUTPUT_ADAPTER.md para saber quais versões existem e quais plataformas cobrir no calendário.
- **Nota visual:** sempre incluir para formatos visuais (carrossel, Reels, TikTok) — mesmo que genérica. Facilita o briefing de design.
